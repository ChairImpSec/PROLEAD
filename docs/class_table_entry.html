Detailed Description

Represents an entry in a contingency table, containing a key and associated data.

The TableEntry class is used to store an observation set (key) and its corresponding counters (data) in a table. Each entry consists of a key that identifies a particular observation and data that counts the occurrences of that observation for each group.
