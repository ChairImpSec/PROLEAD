Detailed Description

template<typename BucketContainer>
class hardware::stats::TableBucket< BucketContainer >

Defines one bucket.

This class defines the bucket, i.e. a list of entries. We repeat that the data type of a bucket (BucketContainer) can be exchanged.

Increments one entry of the bucket.

For every simulated key we either increment the counter of an existing entry or create a new entry. This depends on whether the key is already present in the bucket, i.e. is associated with an entry, or not. However, if a new entry is created, it stil has to be incremented.

Parameters
keyThe simulated key.
number_of_groupsThe number of groups, i.e. the number of counter a new entry must encompass.
group_indexThe index of the group, i.e. the counter we need to update.
size_of_key_in_bytesThe byte-size of the key which is required to compare the key with the entry-keys.
