Searches for a probe in a vector by signal index and clock cycle.

This function searches a vector of probes for a specific probe that matches the given signal index and clock cycle. If the probe is found, its index within the vector is returned. If the probe is not found, a runtime error is thrown.

Parameters
signal_indexThe signal index of the probe to search for.
clock_cycleThe clock cycle of the probe to search for.
probesA vector containing Probe objects.
Returns
The index of the probe within the vector.
Exceptions
std::runtime_errorIf the probe is not found in the vector.
