Detailed Description

This class defines a probe that records the stable signal driven by one wire during a single clock cycle. A probe can be extended based on a probe-extension procedure in order to get a list of probe extensions. The difference to the probe-extension classes is that all probes have a notion of time, i.e. the clock cycle in which they record.
