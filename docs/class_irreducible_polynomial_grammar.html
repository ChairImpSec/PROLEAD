Detailed Description

Grammar definition for parsing irreducible polynomials.

This grammar is used for parsing irreducible polynomials with terms of the form "ax^b" where "a" and "b" are unsigned integers. The grammar supports positive integer coefficients for monomials, optional exponents for the variable "x", and addition of multiple monomials to form a polynomial. For example "3x^2 + 2x + 1" is parsed into a vector of pairs: {(3, 2), (2, 1), (1,0)}.
