Detailed Description

Represents a truth table with a variable amount of input columns and 2 output values F(Toggle), G(Glitch)

bool Hardware::TruthTable::IsEquivalentToOperation ( Operation< CustomOperation > &  op,

Checks if The operation and table return the same output for all possible inputs The input for the operation will be mutated, because: The table uses order: a b c ... a' b' c' ... g_a g_b g_c .. While Operation uses a' a g_a b' b g_b ...

Parameters
opOperation to compare to. Should use ordering a' a g_a b' b g_b ...
for_glitchif true will compare op to the G (glitch) output of table otherwise for F (toggle)
