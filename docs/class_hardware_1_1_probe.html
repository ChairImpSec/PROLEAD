Detailed Description

This class defines a probe that records the stable signal driven by one wire during a single clock cycle. A probe can be extended based on a propagation procedure in order to get a list of Probe elements. The difference to the Propagation class is that all probes have a notion of time, i.e. the clock cycle in which they record and that they do not store the propagation procedure.
