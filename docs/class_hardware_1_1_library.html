Evaluates a particular output function of a particular cell based on the given input values.

The input values are provided in a bitsliced fashion. Every input values represents a single bit of 64 different inputs.

Parameters
cell_indexThe index of the cell.
output_indexThe index of the output function.
input_valuesThe (bitsliced) input values for the cell.
Returns
The (bitsliced) output value of the output function.
